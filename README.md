## Installation

Installez les bibliothèques dont vous avez besoin avec :

```bash
pip install <nom_du_package>
```

## Lancer le dashboard

Selon votre environnement, utilisez `python` ou `python3` :

```bash
python -m streamlit run app.py
# ou
python3 -m streamlit run app.py
```
